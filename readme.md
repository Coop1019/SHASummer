# SHASummer

SHASummer 2.0 is a macOS app that calculates the SHA-256 hash of a selected file and copies it to the clipboard. When a file is opened with the application, it copies the SHA-256 hash and then automatically quits.

### Installation Instructions:
1. Download the application from the Releases tab
2. Add the downloaded application to the "Applications" folder
3. If you want the app to open automatically for a file type, such as `.xpl` files in X-Plane:
4. Navigate in Finder to a file of your chosen type.
5. Right click and choose "Get Info".
6. Under "Open with:" choose "Other" and then click "SHASummer".
7. Press "Change All" to associate it with the file type as the default application.
